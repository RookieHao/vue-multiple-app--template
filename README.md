# multiple-pages-app

```
  多应用分包构建，一次构建多应用也可单独构建。
  应用构建部署是publicPath和代理地址serviceURL在pathConfig.js文件中配置
  项目已配置优化功能包含cdn引入外部库，骨架屏配置，svg雪碧图，gzip压缩。
  有其他需求可自由添加
```
